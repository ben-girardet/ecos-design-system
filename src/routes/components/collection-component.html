Collection

The collection component is a flow component. It handles how several child components are displayed together. This component is well suited to display a list of content such as a list of articles or cards.

Button 1 Button 2 Button 3 Button 4

You can adjust the minimum width of each element in the collection with the `min-item-width` property. Each item will then be displayed organized in a grid where each element is at least this width and as many elements as possible can fit in the container width.
