Components Calendar

Calendar

Display a monthly calendar view and allow user to pick a date

Single Mode

In its default form, a calendar is design to represent or pick a single date.

Range Mode

With `mode="range"` we can set the calendar to represent or pick a range of dates.

Multiple Mode

And with `mode="multiple"` it can represent or pick multiple separate dates

Min and max dates

With `min="..."` and max="..."` the calendar will force the user to choose between these constraints. Its possible to set only `min` or only `max`.

Disabled dates

With `disabled-week-days="1,2"` its possible to disable week-days and with `disabled-dates="..."` one can disabled a list of specific dates.

Appearance

By default the appearance is set to `neutral` but it can be changed to `lightweight`.

Set original month and year

By default the calendar will display the current month and year, but this can be change with its respective attributes

Year selector range

By default the calendar offers a range between current year - 50 and current year + 2. This can be change with the `min-year` and `max-year` attributes.
