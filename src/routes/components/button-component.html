Button

Web component implementation of an HTML button element. The component supports several visual appearances (accent, lightweight, neutral, outline, stealth).

The ecos version is slightly extended from its FAST version. There is a padding added horizontally when the corner-radius is bigger than 10px to allow for the button content to breath a little more around the edges.
