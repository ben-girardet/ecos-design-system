Next

Documentation in progress

The Next component offers the possibility to create several "screens" that can be navigated to with "previous" and "next" effect.

This kind of effect is often used in settings screens and such.

Navigation options

  • next() : go to the next screen according to the DOM order
  • prev() : go to the prev screen according to the DOM order
  • back() : navigate back to the previously displayed item (if any)
  • goToId(id: string, direction: 'prev' | 'next' | 'auto') : go to a specific DOM id
  • hasHistory: boolean saying if the Next element has an item to go back to

Fixed usage

The fixed usage creates a Next element that will completely fill the available space (100% width and 100% height)

Default usage

The default usage create a Next element that is "as big as necessary". In order to control a width or height, it can be attributed by styling either the `ecos-next` element or its children.
